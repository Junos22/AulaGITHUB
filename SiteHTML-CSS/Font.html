Tag Font

A tag Font permite ao desevolvedor escolher o tipo, o tamanho e a cor da fonte de um texto. Os elementos responsáveis por isso, respectivamente, são: face, size e color.

O elemento face indicará qual a fonte a ser utilizada, como Tahoma, Arial etc.

O elemento size varia entre 1 e 7, sendo o valor 3 o padrão para a maioria dos browsers.

O elemento color define a cor da fonte, podendo ser definida através de números hexadecimais ou através de nomes de cores previamente definidos.
