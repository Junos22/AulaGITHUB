O que é HTML?

HTML (Hiper Text Markup Language) é a linguagem de programação voltada para a criação de sites. É a linguagem responsável pela parte gráfica do site (layout do site), sendo portanto uma linguagem do tipo front-end. Apesar do HTML estar voltada à parte gráfica do site, sua abrangência é um tanto quanto limitada, não sendo possível trabalhar-se com gráficos e animações, por si só. Entratanto, a linguagem HTML permite a associação com outras linguagens, permitindo suprir todas as necessidades do desenvolvedor para a criação de sites.
