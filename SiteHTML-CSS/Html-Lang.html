Tag html

A tag html é o comando que descreve que tudo o que será contido entre suas tags de abertura e fechamento devam ser entendidos como um documento html. Sem esta tag, a criação do site é impossível e tudo seria entendido como um texto aleatório sem função alguma em programação.



Elemento Lang

O elemento lang faz parte da tag html e é utilizada para se especificar qual é o tipo de idioma que o site irá conter. Este elemento em nada muda a maneira de como se programa em HTML. Ele apenas normatiza o idioma, facilitando-se a detecção do idioma pelos browsers (para tradução do conteúdo do site), assim como facilita buscadores para a entrega do site para determinado público-alvo. O elemento lang é utilizado dentro da tag html e segue a nomenclatura ISO. O primeiro valor do elemento refere-se à língua de determinado(s) país(es) e o segundo valor delimita de qual país aquele idioma se refere especificamente. Pode-se utilizar apenas o primeiro valor, sem especificar o país de origem, dependendo-se do intuito a ser buscado.
